# 技術スタックとか使用技術

【フロントエンド (Framework)】
- Flutter (Dart) - クロスプラットフォームのUI構築

【バックエンド・インフラ (BaaS)】
- Supabase (supabase_flutter) - ユーザーデータや日記データの保存、バックエンド構築
- Firebase Hosting - Webアプリとしてのデプロイ・ホスティング

【AI（生成AIエンジン）】
- Google Gemini API (google_generative_ai) - ユーザーの入力テキストからポジティブな日記（コイン）を生成

【物理エンジン・アニメーション】
- Flame / Forge2D (flame, flame_forge2d) - ブタの貯金箱内のコインの物理演算、重力シミュレーション
- Sensors Plus (sensors_plus) - スマホの傾き（ジャイロセンサー）の取得と重力への反映

【UI・機能拡張パッケージ】
- FL Chart (fl_chart) - 感情分析やコイン取得状況のグラフ化
- Share Plus / Screenshot (share_plus, screenshot) - 画面のスクリーンショット撮影とSNS（Xなど）へのシェア機能
- Shared Preferences (shared_preferences) - ローカルでの一時的なデータ保存
- Flutter Dotenv (flutter_dotenv) - APIキーなどの環境変数の安全な管理
